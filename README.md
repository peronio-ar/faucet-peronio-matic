# La Crypta Peronio & Matic Faucet

- [web3modal](https://github.com/Web3Modal/web3modal/)
- [ethers](https://docs.ethers.io/v5/)

## Usage

1. Install packages

```bash
yarn
```

2. Copy .env example

```bash
cp .env.example .env
```

Set REACT_APP_FAUCET_CONTRACT with your Faucet Contract Address

3. Start the app

```bash
yarn start

# OR

yarn dev
```
